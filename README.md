## Sử dụng web:

Step 1: Tạo file .env ở trong thư mục chứa source code(Gửi kèm theo trong link báo cáo)

Step 2: Cài đặt các gói cần thiết
```
npm install
```

Step 3: Khởi chạy chương trình

```
npm start
```
